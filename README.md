## How to import

### n8n
1. Open your n8n instance ([cloud](https://n8n.io) or self-hosted)
2. Go to **Workflows → Import from File** (or paste JSON)
3. Upload the `.json` file
4. Configure credentials (Gmail, Google Sheets, Slack) and activate

### Make (Integromat)
1. Log in at [make.com](https://www.make.com)
2. Go to **Scenarios → Import Blueprint**
3. Upload the `.json` file
4. Connect your apps and click **Save**
